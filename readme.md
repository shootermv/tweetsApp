# tweets app — the BackboneJS app
Little single page appllication for display top 10 tweets from Hacker News.
Each article may have nested comments - which being dispayed when user clicks the article 
### The frameworks used:
* 1.responsive design in html5/css3 
* 2.AMD - for load the js modules dynamically
* 3.backbonejs -as javascript framework
 
### Running the app

You can run this application at any server (e.t. nodejs, Apache, VS)
